# BrandCore — Site Web

Site statique multi-pages pour **BrandCore**, une agence de communication et relations publiques basée à Cotonou, Bénin.

## Stack

- HTML / CSS statique (aucun framework)
- Polices Google Fonts : Poppins + Inter
- Pas de dépendances npm

## Lancer le site

```bash
python3 -m http.server 5000 --directory 'BrandCore (4)/BrandCore'
```

Le site est accessible sur le port **5000**.

## Structure des fichiers

```
BrandCore (4)/BrandCore/
  index.html      — Page d'accueil
  services.html   — Services
  methode.html    — Approche / Méthode
  apropos.html    — À propos
  contact.html    — Contact
  style.css       — Styles globaux
  logo.png        — Logo de l'agence
```

## Autres fichiers

- `BrandCore/` — Version précédente du site (sans logo)
- `Brand_Core_Charte_Graphique_Premium-1.docx` — Charte graphique
- `Fiche_de_constitution_BrandCore_avec_page_de_garde.docx` — Fiche de constitution
- `aurel.png`, `giovani.png`, `sunshi.png` — Photos d'équipe

## Préférences utilisateur
