# BrandCore

Site vitrine statique de BrandCore.

## Déploiement

Ce dépôt est prêt pour un déploiement statique sur Netlify, Vercel ou GitHub Pages.

- Page d'accueil : index.html
- Fichier de configuration Netlify : netlify.toml

## Lancer localement

Ouvrez index.html dans votre navigateur ou servez le dossier avec un simple serveur statique.
